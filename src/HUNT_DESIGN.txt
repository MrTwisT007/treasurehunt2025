Layer 2:
- A mixture of text and hex data in a text file. Text is some lore to keep intrigue going. Following HEX dump should be converted into a file using a hex editor. The file shall be an image with layer 3 obfuscated by being placed after the image's EOI
- Essentially a dual-type file: a jpeg with an zip tacked on to the end - can be viewed as a jpeg, but can also be opened with 7zip
- A hint will be provided to look for magic numbers, which should hint at what type of file it is to start

Layer 3:
- This should be the final and hardest layer to solve. A red herring of some sort must be present to misdirect. Right now the idea is that it would be a literal image of a red herring
- An archive with three files in it: FILE1, FILE2 and FILE4. The "missing" FILE3 shall be hidden
- FILE3 shall be a 7z archive. It shall be passworded and the contents shall be the steam keys. This file shall not be visible in the archive and be tacked onto one of the other files.
- FILE1 shall be a zip file with an image tacked on after EOI. The archive will be most visible since it can be opened immediately and shall contain ciphertext with a message and password for FILE3. The image shall contain the cipherkey for the text in the archive
- FILE2 shall contain the red herring after the EOI and serve purely to distract
- FILE4 shall be an image with a text printout of hex tacked onto the end. This would require double conversion from hex to ascii, which is hex again. This will be FILE3

Key concepts:
- Payloads should be obfuscated such that the unlocking a layer should reveal the next layer entirely. Payloads must be obfuscated to prevent snooping with a hex editor and directly copying the contents without properly solving the layer
- Steganography will be used as the key element for the hunt.
- An introduction shall accompany the first layer. It will be a mock story of encountering some sort of deep conspiracy that goes several layers deep. The first layer is provided by the whistleblower who's seen whats at the bottom, but the recipient must solve for himself for "reasons (tbd, e.g. the whistleblower is complicit and had to hide it himself)". A hint to the solution of the first layer and general concepts shall be included here.
- Each layer should also provide some sort of initial direction and starting hint. A continuation of the "lore" will also be included in parts.
- Hunt must be largely solvable without requiring any specific tooling, so generic hex editors, well known cyphers, etc. Steganography is limited this way so more cool things are not allowed (e.g. zip file shenanigans with central directory shifting and hiding binary data blobs)
- As a bonus, after solving the puzzle the source to the whole project shall be provided.
